서비스 이용약관

제 1 조 (목적)

본 약관은 "H4N"(이하 "운영자")이 제공하는 인공지능 기반 챗봇 서비스 "HLS"(이하 "서비스")를 이용함에 있어, 운영자와 이용자 간의 권리, 의무, 책임 사항 및 데이터 투명성에 관한 필수 규정을 명시함을 목적으로 합니다.

제 2 조 (용어의 정의)

1. "이용자"란 본 약관에 동의하고, 운영자가 허가하거나 발급한 고유 인증 키(Access Key)를 입력하여 서비스에 접속하는 자를 말합니다.

2. "고유 인증 키"란 인가되지 않은 외부 사용자의 무단 트래픽을 통제하기 위해 이용자가 헤더 또는 파라미터 형태로 전송해야 하는 문자열 식별자를 의미합니다.

3. "로그 데이터"란 이용자가 서비스를 호출하는 과정에서 발생하는 암호화 가웃된 키, 요청 시각, 대화 내용 전반을 의미합니다.

제 3 조 (약관의 효력 및 동의)

1. 본 약관은 이용자가 서비스 웹 인터페이스 상에서 고유 키를 입력한 뒤 서버 인증 요청을 전송하여 정상 진입하는 즉시 동의한 것으로 간주하며, 그 시점부터 효력이 발생합니다.

2. 운영자는 서비스 성능 최적화 또는 보안 환경 규격 변경에 따라 본 약관을 개정할 수 있으며, 개정된 약관은 본 페이지에 업데이트함으로써 공지를 갈음합니다.

제 4 조 (이용자의 대화 및 영구 기록에 대한 공지)

운영자는 시스템의 기술적 안정성 보장 및 추후 보안 관제 연구를 위해 이용자의 사용 패턴과 연동 로그를 상시 모니터링합니다.

[데이터 영구 적재 고지]
본 서비스는 사용자가 입력창을 통해 전송한 모든 질문 프롬프트(대화 내용), 사용자가 인증에 사용한 고유 인증 키, 정확한 접속 타임스탬프(KST)를 클라우드 인프라 내부 백엔드 스토리지(KV Store)에 필터링 없이 영구적으로 적재 및 보관합니다.

이용자는 이와 같은 자동 수집 메커니즘을 명확히 인지해야 하며, 본인의 개인정보, 사생활 데이터, 또는 기업 및 연구 보안에 위배되는 민감 정보 및 소스코드를 입력창에 포함하지 않을 전적인 책임이 있습니다.

제 5 조 (외부 API 연동 및 데이터 전송)

1. 본 서비스는 독자적인 자연어 추론 결과물 도출을 위해 외부 분산 인프라 생태계(NVIDIA API)를 차용하여 추론 파이프라인을 운영하고 있습니다.

2. 이용자가 입력한 텍스트 데이터 패킷은 인공지능 가속 연산을 위하여 실시간으로 외부 연동 인터페이스 서버(NVIDIA 인프라스트럭처 호스트)로 안전하게 터널링 및 암호화 전송됩니다.

제 6 조 (서비스의 한계 및 책임 제한)

1. 본 서비스는 대형 생성형 모델(LLM)을 기반으로 동작하므로 기술적 한계에 의한 환각 현상(Hallucination), 허위 정보 유출, 혹은 문맥 왜곡 현상이 발생할 수 있습니다. 생성된 결과물의 활용 여부는 이용자 본인의 지적 판단에 위임하며, 이로 인한 정신적·물질적 오차나 손해에 대하여 운영자는 어떠한 법적 배상 책임도 지지 않습니다.

2. 외부 API 통신 장애, 호스트 서버 점검, 네트워크 백본 라우팅 유실 등 불가항력적인 원인으로 인한 서비스 공급 중단에 대해서도 면책됩니다.

시행 일자: 2026년 7월 11일

개인정보 처리방침

1. 처리하는 데이터 수집 항목

서비스 내부 시스템 텔레메트리 파이프라인 및 백엔드 라우터는 회원 가입 절차를 대체하여 사용자 권한 유효성을 식별하고 비인가 공격을 탐지하기 위해 아래와 같은 환경적 데이터를 자동으로 적재합니다.

  • 이용자가 전송 시 헤더에 명시한 고유 접근 인증 키(X-Access-Key)
  • 네트워크 세션 인입 및 API 요청 처리가 수행된 타임스탬프
  • 입력창에 직접 타이핑하여 전송한 문장 데이터(User Prompt)

2. 데이터의 수집 및 이용 목적

적재되는 데이터 파이프라인은 상업적 마케팅 목적으로 절대 활용되지 않으며, 오직 기술적 목적에 집중하여 처리됩니다.

  • 헤더 대조 알고리즘을 통한 비인가 IP 및 크래킹 시도 차단
  • API 요청 시 데이터 규격 미준수로 인한 컴파일/런타임 디버깅 및 예외 처리 추적
  • 보안 및 취약점 분석, 악성 유해 페이로드 유입 차단을 위한 입력 검증 연구 목적

3. 데이터의 보유 및 보관 기간

[보유 정책 정보]
수집된 모든 사용 이력 및 데이터 덤프 항목은 서버 자원이 허용하고 서비스 가동 체계가 유지되는 한 영구 적재 파이프라인 보관을 원칙으로 유지합니다.

단, 이용자가 본인의 인증 자격 증명을 소명하여 로그 소거 처리를 명시적으로 청구하거나 시스템 전체 구조 마이그레이션이 발생할 시에는 영구 적재본이 파기될 수 있습니다.

4. 데이터의 위탁 처리 안내 (제3자 데이터 전송)

본 서비스는 지능형 언어 응답 스펙을 충족하기 위하여 데이터 연산 처리를 아래의 전문 기관에 안전하게 위탁하여 처리하고 있습니다.

  • 위탁받는 자 (수탁기관): NVIDIA Corporation
  • 위탁 목적: z-ai/glm-5.2 인공지능 언어 모델의 실시간 추론 연산 대행
  • 위탁 항목: 사용자가 채팅 콘솔에 입력한 순수 텍스트 대화 데이터 히스토리

5. 이용자의 유의사항

본 시스템에 인입된 모든 쿼리는 자동 영구 저장소에 실시간 캐싱되므로, 패스워드 원문, 개인 주민등록식별 코드, 미공개 보안 취약점 소스코드 등 유출 시 보안적 데미지가 발생할 가능성이 다분한 데이터 원형은 절대 전송하지 마시기 바랍니다.
